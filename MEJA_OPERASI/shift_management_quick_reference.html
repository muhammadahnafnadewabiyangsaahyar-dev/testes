v_jadwal_shift_harian

VIEW

Daily shift schedule with employee details, shift info, and confirmation status.

SELECT * FROM v_jadwal_shift_harian WHERE tanggal_shift >= CURDATE() ORDER BY tanggal_shift;

v_absensi_dengan_shift

VIEW

Attendance records with shift information, work duration, and overwork calculation.

SELECT * FROM v_absensi_dengan_shift WHERE user_id = 1 AND tanggal_absensi >= '2025-01-01';

v_ringkasan_gaji

VIEW

Payroll summary with all components, deductions, and net salary.

SELECT * FROM v_ringkasan_gaji WHERE periode_bulan = 1 AND periode_tahun = 2025;

üÜò Troubleshooting

Foreign Key Error

Cause: Referenced data doesn't exist

Solution: Ensure parent records exist before inserting child records

Duplicate Entry Error

Cause: Unique constraint violation

Solution: Check for existing records before insert or use ON DUPLICATE KEY UPDATE

Trigger Not Working

Cause: Trigger might have syntax error

Solution: Drop and recreate trigger, check SHOW TRIGGERS output
